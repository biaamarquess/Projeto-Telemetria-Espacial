# Projeto-Telemetria-Espacial 🚀

# Explicação do Projeto
Este projeto desenvolve um sistema de verificação para o pré-lançamento da nave Aurora Siger, analisando dados de telemetria para autorizar ou abortar a decolagem. O processo foi automatizado em Python, incluindo análise energética e identificação de falhas, além de considerar aspectos éticos e de sustentabilidade no uso da tecnologia.

# O que o sistema verifica:
- Temperatura interna e externa
- Integridade estrutural (se está ok ou com falha)
- Nível de energia da bateria]
- Pressão dos tanques de combustível
- Status dos módulos críticos

# Como Executar o Código
- Salve o arquivo
- Abra o terminal
- Execute o comando: python telemetria_espacial.py
