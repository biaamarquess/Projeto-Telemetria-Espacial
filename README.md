# Projeto-Telemetria-Espacial 🚀

# Explicação do Projeto
Este projeto é um sistema de verificação de telemetria para lançamentos espaciais. Ele simula a leitura de dados de sensores de um foguete e decide se ele está PRONTO PARA DECOLAR ou se a DECOLAGEM DEVE SER ABORTADA.

# O que o sistema verifica:
- Temperatura interna e externa
- Integridade estrutural (se está ok ou com falha)
- Nível de energia da bateria]
- Pressão dos tanques de combustível
- Status dos módulos críticos

# Como Executar o Código
- Salve o arquivo
- Abra o terminal
- Execute o comando: python telemetria_espacial.py
